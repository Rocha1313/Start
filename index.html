Visual Studio Code (VSCode) is a versatile code editor used for many programming languages, most commonly with JavaScript and its variants.

VSCode

Node.js is a JavaScript runtime built on Chrome's V8 JavaScript engine. While it might not be utilized immediately in early classes, it's crucial for more advanced coding and becomes increasingly important as the course progresses.

NodeJS

Additional plugins to install for VSCode:

  • GitLens (Required at Start)
  • ESLint (Required at Start)
  • Explorer Exclude (Required at Start)
  • JavaScript (ES6) code snippets (Required at Start)
  • Polyglot Notebooks (Required at Start)
  • Live Server (Install when instructed)
  • Prettier (Install when instructed)
  • vscode-styled-components (Install when instructed)
  • Choose a theme that you like (Example: Doki Theme)
